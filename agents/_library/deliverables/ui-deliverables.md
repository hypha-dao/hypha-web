### UI Deliverables

| Artifact             | Description                                                                                 |
| -------------------- | ------------------------------------------------------------------------------------------- |
| **Components**       | React components in `@hypha-platform/ui` (primitives) or `@hypha-platform/epics` (features) |
| **Stories**          | Storybook `.stories.tsx` files demonstrating component states and variants                  |
| **Design Tokens**    | CSS custom properties in `packages/ui-utils/src/theme/`                                     |
| **Responsive Specs** | Components working across breakpoints (sm, md, lg, xl)                                      |
| **Dark Mode**        | Components rendering correctly in both light and dark themes                                |
| **Accessibility**    | ARIA attributes, keyboard navigation, screen reader support                                 |
| **Hooks**            | Client-side data hooks in `@hypha-platform/epics/src/{domain}/hooks/`                       |
