### Test Deliverables

| Artifact             | Description                                                                  |
| -------------------- | ---------------------------------------------------------------------------- |
| **Test Plan**        | Scope, approach, environments, and coverage targets per feature              |
| **E2E Specs**        | Playwright specs with page objects in `apps/web-e2e/src/`                    |
| **Contract Tests**   | Hardhat test files with fixtures and helpers in `packages/storage-evm/test/` |
| **Unit Tests**       | Vitest specs co-located with source code                                     |
| **Bug Reports**      | Steps to reproduce, expected vs actual, severity, affected domain            |
| **Coverage Report**  | Test coverage analysis identifying gaps in critical paths                    |
| **Regression Suite** | Curated set of tests that must pass before any release                       |
