### QA & Test Engineering

- **Test Strategy Design:** Defining test pyramids, coverage targets, risk-based testing priorities, and environment strategies for web applications
- **E2E Test Automation:** Designing and maintaining end-to-end test suites using browser automation frameworks (Playwright, Cypress) with resilient locator strategies, parallel execution, and CI/CD integration
- **Component & Unit Testing:** Isolating units of behavior with Vitest/Jest, testing React components with Testing Library, and maintaining fast feedback loops
- **Accessibility Testing:** Automated WCAG compliance scanning (axe-core), assistive technology compatibility validation, and inclusive design review
- **Visual Regression Testing:** Screenshot comparison workflows, baseline management, cross-browser/cross-platform snapshot consistency, and deterministic rendering strategies
- **Performance Testing:** Core Web Vitals measurement, Lighthouse CI integration, load testing, and runtime performance profiling
- **Security Testing Awareness:** OWASP Top 10 verification, input validation testing, authentication/authorization flow coverage, and CSP/header verification
- **Test Data Management:** Fixture design, factory patterns, database seeding strategies, and test isolation through controlled data states
