### User Research Deliverables

| Artifact                    | Description                                                                                          |
| --------------------------- | ---------------------------------------------------------------------------------------------------- |
| **Research Plan**           | Study objectives, research questions, methodology, participant criteria, timeline, and success metrics |
| **Interview Guide**         | Semi-structured question set with probes, organized by topic area, with warm-up and closing scripts   |
| **Usability Test Script**   | Task scenarios, pre/post-test questions, facilitator instructions, and success criteria per task      |
| **Survey Instrument**       | Validated questionnaire with screening, core, and demographic sections; includes analysis plan        |
| **Persona Profiles**        | Evidence-based user archetypes with goals, pain points, behaviors, context, and supporting quotes     |
| **Journey Map**             | End-to-end visualization of user experience across phases, touchpoints, emotions, and opportunities   |
| **Usability Report**        | Findings ranked by severity, with evidence (clips, quotes, metrics), and prioritized recommendations |
| **Insight Repository Entry**| Atomic research nuggets tagged by theme, method, date, and confidence level for cross-study reuse     |
| **Affinity Diagram**        | Clustered research observations organized into themes and patterns for collaborative sensemaking      |
| **Jobs-to-be-Done Profile** | Job statement, functional/emotional/social dimensions, current solutions, and switching triggers      |
| **Heuristic Evaluation**    | Issue log with heuristic violated, severity rating, location, screenshot, and fix recommendation      |
| **Research Debrief Deck**   | Stakeholder-ready presentation synthesizing key findings, implications, and recommended next steps    |
